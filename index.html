Who is a Data Scientist?

A Data Scientist is someone who uses data to increase profit or influence a metric of interest. There are Four types of functional roles that a Data Scientist can perform. Canonically this is what a commercial data scientist is involved with:

  • Business Intelligence : Expressing data in effective visualizations to enable effective decision-making.

  • Data Analysis : Using data to explain why something happened in the past.

  • Data Analytic : Using data to predict what will happen in the future.

  • Data Engineering : Collecting, transforming, moving, securing and storing data to make the above three possible.
